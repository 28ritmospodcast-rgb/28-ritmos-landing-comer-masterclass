# 28 Ritmos

Proyecto web, marketing y base de conocimiento para 28 Ritmos.

## Empezar

Para retomar el proyecto en un chat nuevo, leer primero:

- `base-28-ritmos/README.md`
- `base-28-ritmos/00-empezar-nuevo-chat.md`

## Páginas públicas

- Come con tu Ciclo: `https://28-ritmos-landing-comer-masterclass.vercel.app/`
- Lee tu Ciclo: `https://28-ritmos-landing-comer-masterclass.vercel.app/lee-tu-ciclo`

## Archivos principales

- `index.html`: landing de Come con tu Ciclo.
- `lee-tu-ciclo.html`: landing de Lee tu Ciclo.
- `styles.css`: estilos compartidos.
- `marketing/`: emails, narrativa y materiales de campaña.
- `base-28-ritmos/`: contexto estable para nuevos chats.
