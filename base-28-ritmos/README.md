# Base 28 Ritmos

Este repositorio interno existe para que cada nuevo chat o colaborador pueda entender rápido el contexto de 28 Ritmos sin reconstruir todo desde cero.

## Cómo usarlo en un nuevo chat

Al empezar, pegar o adjuntar primero:

1. `00-empezar-nuevo-chat.md`
2. El archivo especifico del trabajo que quieras hacer:
   - Para web: `05-web-y-deploy.md`
   - Para emails, ads o copy: `02-voz-y-copy.md` y `03-ofertas-y-links.md`
   - Para diseño: `04-estetica-y-diseno.md`
   - Para ubicar archivos: `06-activos-y-archivos.md`

## Mapa de archivos

- `00-empezar-nuevo-chat.md`: resumen corto para pegar al inicio de cualquier chat.
- `01-contexto-proyecto.md`: qué es 28 Ritmos, para quien es y que problema trabaja.
- `02-voz-y-copy.md`: voz, tono, palabras, límites y criterios de comunicacion.
- `03-ofertas-y-links.md`: productos, precios, links y posicionamiento.
- `04-estetica-y-diseno.md`: guía visual condensada.
- `05-web-y-deploy.md`: páginas, archivos, tracking, deploy y reglas técnicas.
- `06-activos-y-archivos.md`: dónde están imágenes, PDFs, branding y documentos.
- `07-checklist-nuevo-chat.md`: checklist para arrancar tareas nuevas.
