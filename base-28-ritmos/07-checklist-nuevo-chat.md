# Checklist Para Nuevo Chat

## Si el trabajo es de copy

- Confirmar producto: Lee tu Ciclo, Come con tu Ciclo o Método 28 Ritmos.
- Revisar `02-voz-y-copy.md`.
- Revisar `03-ofertas-y-links.md`.
- Usar voseo y acentos.
- Nombrar dolores especificos sin atacar a la lectora.
- Evitar promesas médicas o urgencia falsa.

## Si el trabajo es de diseño/web

- Revisar `04-estetica-y-diseno.md`.
- Revisar `05-web-y-deploy.md`.
- Mantener estilo editorial, organico, calido.
- Revisar mobile.
- Optimizar imágenes.
- No agregar elementos que se sientan corporativos o wellness startup.

## Si el trabajo es de ads

- Mantener primera linea clara.
- Hablar de transformación o verdad especifica.
- No hacer claims medicos fuertes.
- No usar vergüenza corporal ni before/after agresivo.
- Separar angulos:
  - comida y culpa;
  - hambre y antojos;
  - energía cíclica;
  - fertilidad y ventana fértil;
  - perimenopausia;
  - dejar de depender de la app.

## Si el trabajo es de email

- Mantener tono de carta.
- No vender demasiado pronto sí el correo abre una historia sensible.
- Usar preguntas que inviten a responder.
- Skool puede aparecer cómo espacio de comunidad.
- Come con tu Ciclo puede desbloquearse en Skool al nivel 3.

## Si se va a publicar web

1. Revisar `git status --short`.
2. Editar solo archivos necesarios.
3. Revisar diff.
4. Commit claro.
5. Push a `main`.
6. Esperar deploy de Vercel.
